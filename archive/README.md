# Archive 디렉토리

이 디렉토리는 프로젝트의 이전 버전 코드를 보관합니다.

## v1.0 (정상 작동 확인된 원본)

`v1.0/app.py`는 리팩터링 이전의 단일 파일 Flask 서버입니다.

### 특징
- 단일 파일 구조 (1270줄)
- 모든 기능이 하나의 파일에 포함
- 실제 운영 환경에서 정상 작동 확인
- 테스트 통과: `tests/archive/test_complete.py`

### 사용 목적
- 리팩터링 기준점 (baseline)
- 버그 발생 시 참조
- 기능 검증

### 실행 방법
```bash
# 주의: 이 코드는 참조용입니다. 실행하지 마세요.
# 대신 리팩터링된 server/app.py를 사용하세요.
```
