WCMS 소개

Woosuk Computer Management System - 실습실 PC 원격 관리 시스템

프로젝트 개요

WCMS는 실습실 PC를 중앙에서 효율적으로 관리하고 모니터링하기 위한 오픈소스 원격 관리 시스템입니다. Windows 서비스 기반 클라이언트와 Flask 웹 서버를 통해 실시간 PC 상태 모니터링, 원격 명령 실행, 프로그램 설치/삭제, 계정 관리 등을 제공합니다.

주요 기능

📊

실시간 모니터링

CPU, RAM, 디스크 사용률 실시간 추적

⚡

원격 제어

종료, 재시작, 로그오프 원격 실행

📦

프로그램 관리

Chocolatey 기반 자동 설치/삭제

👥

계정 관리

Windows 계정 생성/삭제/비밀번호 변경

🔒

보안

PIN 인증, HTTPS, Rate Limiting

🔄

자동 업데이트

클라이언트 자동 버전 체크 및 업데이트

  • SQLite (WAL 모드)
  • bcrypt, Flask-Talisman

Frontend
